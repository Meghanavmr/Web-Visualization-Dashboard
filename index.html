    • Summary: Latitude vs. X

    ***The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMapi API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlip to plot various aspects of the weather vs. latitude. Factoes we looked at included: temperature, cloudiness, wind speed and humidity. This also provides the source data visualizztions as part of the analysis as well as explanations and descriptions of any trends and correlations wtinessed.***
